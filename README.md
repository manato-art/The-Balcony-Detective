# ベランダ探偵

マンションのベランダ・洗濯物・生活感だけを見て「誰が住んでるか」を当てる、飲み会向けスマホ推理ゲーム（MVP: 通常クイズモード）。

## 起動

```bash
node server.js
# → http://localhost:5610 （スマホは同一Wi-FiでLAN IPにアクセス）
```

サーバー無しで `index.html` をブラウザで直接開いてもOK（依存ゼロ・完全オフライン動作）。

## 遊び方

1. プレイヤー登録（1〜8人）＋1人あたりの回数（1〜5回）
2. マンションを選ぶ（10タイプ）
3. スマホを回して「◯◯号室の住人を当てろ」
4. ベランダのヒント→必要なら追加調査（ポスト=発覚率50% / 待つ=ランダムイベント / 隣人=嘘かも）
5. 4択回答。「自信満々」宣言で+50点、外したら二口
6. 全ターン終了で結果発表＋偏見王の審判

## 構成

- `js/data/residents.js` — 住人33種（普通/盛り上がり/罠/レア枠）・ヒント・煽り文
- `js/data/mansions.js` — マンション10種と住人の出現重み
- `js/data/events.js` — ランダムイベント・飲みルール
- `js/game.js` — ゲームロジック（UI非依存）
- `js/ui.js` / `css/style.css` — スマホ縦画面UI
- `test/validate.js` — データ整合性＋3000ターンのシミュレーション検証

## 検証

```bash
node test/validate.js
```
